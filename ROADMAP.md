# Draw & Guess — план развития

> Полный роадмап продукта. Фазы идут по приоритету: удержание → виральность →
> мультиплеер → монетизация.

## 🔴 Фаза 1 — Удержание (люди возвращаются)

### Ежедневный челлендж
- Каждый день одно и то же слово для всех игроков
- Счёт дня, история попыток
- «Ты #47 из 1 203 игроков сегодня»
- Даёт причину заходить каждый день

### Глобальная таблица лидеров
- Топ за сегодня / неделю / всё время
- Никнейм + очки + страна флагом
- База данных (Supabase / PlanetScale — бесплатный tier)

### Профиль игрока
- Никнейм, аватар (эмодзи на выбор)
- Статистика: сыграно раундов, угадано, лучший счёт
- Стрик — сколько дней подряд играл
- Хранится в БД, вход через Google/GitHub (NextAuth)

### Достижения
- «Первое угадывание», «10 раундов подряд», «Скорость молнии» (<5 сек)
- Визуальные бейджи на профиле
- Нотификация при получении

## 🟡 Фаза 2 — Виральность (люди приводят друзей)

### Шаринг результата
- Кнопка «Поделиться» → красивая картинка-карточка (canvas → PNG)
- «Я набрал 847 очков в Draw & Guess! Можешь побить?»
- Готовая ссылка на сайт в карточке

### Приватные комнаты
- Создаёшь комнату → код (XKCD42), скидываешь другу
- Оба играете одновременно, но каждый сам по себе
- Сравнение результатов в конце («Ты обогнал Vasya на 120 очков!»)
- Не требует WebSocket — общий seed для слов

### Категории слов
- 🐾 Животные, 🍕 Еда, 🏆 Спорт, 🎬 Фильмы, 🌍 Страны
- Игрок выбирает перед игрой; легко добавлять паки

### Уровни сложности
- Лёгкий (cat, house, sun) / Средний (текущие) / Хардкор (архитектура, мифология)

## 🟢 Фаза 3 — Главная фича (мультиплеер)

### Реальный мультиплеер 1v1
- Два игрока, одно слово одновременно
- ИИ оценивает обоих, кто быстрее угадан — побеждает
- WebSocket через Ably/Pusher (бесплатный tier до 200 соединений)

### Групповые комнаты (Skribbl-режим)
- 2–8 игроков, по очереди рисуют — остальные угадывают текстом
- Стандартный проверенный формат; нужен полноценный WebSocket-сервер

### Рейтинговые матчи
- MMR, сезоны, сезонные скины/награды

## 💰 Фаза 4 — Монетизация

### Косметика (основной источник)
- Скины кисти (неон, мел, акварель)
- Темы канваса (тетрадь, доска, бумага)
- Рамки профиля, кастомные эмодзи-аватары
- 50–200 ₽ за предмет / 500 ₽ за пак

### Подписка Premium (~199 ₽/мес)
- Без рекламы, все категории слов, кастомные слова в комнатах, расширенная статистика

### Реклама для бесплатных
- Баннер между раундами (не во время игры)
- Google AdSense / Yandex РСЯ
- Rewarded ad: «Посмотри рекламу → +10 сек»

### Паки слов
- Тематические: «Аниме», «Игры», «Мемы» — 99 ₽
- Корпоративный: кастомные слова для тимбилдингов
